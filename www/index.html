    工事種別
    直接工事費(円)
    純工事費(円)
    工事原価(円)
    工事価格(円)
    消費税(円)
    工事費(円)
    純工事費 = 直接工事費 + 共通仮設費
    工事原価 = 純工事費 + 現場管理費
    工事価格 = 工事原価 + 一般管理費
    消費税 = 工事価格 * 消費税率
    工事費 = 工事価格 + 消費税
    共通費合計 = 共通仮設費 + 現場管理費 + 一般管理費
    対直接工事費共通費率 = 共通費合計 ÷ 直接工事費 * 100

    共通仮設費
    共通仮設費対象額直接工事費-桁等購入費-鋼橋門扉等工場原価+事業損失防止施設費+無償貸与機械等評価額+共通仮設費対象支給品費等
    処分費等の占める割合(直接工事費に含まれる準備費+準備費に含まれる処分費)/(共通仮設費対象額+準備費に含まれる処分費)
    率計算共通仮設費対象額共通仮設費対象額-直接工事費に含まれる処分費+計上対象処分費
    共通仮設費率共通仮設費率(補正前)*大都市補正+施工地域･工事場所区分補正
    率計算分共通仮設費率計算共通仮設費対象額*共通仮設費率
    率計算ｲﾒｰｼﾞｱｯﾌﾟ経費対象額直接工事費-直接工事費に含まれる処分費+無償貸与機械等評価額+ｲﾒｰｼﾞｱｯﾌﾟ経費対象支給品費等
    ｲﾒｰｼﾞｱｯﾌﾟ経費率ｲﾒｰｼﾞｱｯﾌﾟ経費率(補正前)+施工地域･工事場所区分補正
    率計算分ｲﾒｰｼﾞｱｯﾌﾟ経費率計算ｲﾒｰｼﾞｱｯﾌﾟ経費対象額*ｲﾒｰｼﾞｱｯﾌﾟ経費率
    共通仮設費合計率計算分共通仮設費+率計算分ｲﾒｰｼﾞｱｯﾌﾟ経費+事業損失防止施設費+ｲﾒｰｼﾞｱｯﾌﾟ特別計上額+その他積上共通仮設費
    現場管理費
    率計算現場管理費対象額直接工事費-鋼橋門扉等工場原価+無償貸与機械等評価額+共通仮設費+現場管理費対象支給品費等-直接工事費に含まれる準備費-準備費に含まれる処分費+計上対象処分費
    現場管理費率現場管理費率(補正前)*大都市補正+施工地域･工事場所区分補正+施工時期･工事期間補正
    現場管理費率計算現場管理費対象額*現場管理費率
    一般管理費
    率計算一般管理費対象額直接工事費+共通仮設費+現場管理費-直接工事費に含まれる準備費-準備費に含まれる処分費+計上対象処分費
    一般管理費率一般管理費(補正前)*前払金支出割合補正+契約保証補正
    一般管理費率計算一般管理費対象額*一般管理費率
